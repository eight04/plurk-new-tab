Plurk in new tab
================

[![.github/workflows/build.yml](https://github.com/eight04/plurk-new-tab/actions/workflows/build.yml/badge.svg)](https://github.com/eight04/plurk-new-tab/actions/workflows/build.yml)

在手機版plurk中，從使用者頁面跳轉到噗文後，回到使用者頁面時，會重新載入使用者頁面，導致使用者體驗不佳。此 userscript 會讓噗文在新分頁開啟，避免此問題。
